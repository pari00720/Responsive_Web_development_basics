# HTML Basics

A beginner-friendly repository to learn and practice the fundamentals of HTML.

## 📖 About
This repository contains basic HTML programs and examples that cover the core concepts of HTML. It is created for learning, revision, and strengthening web development fundamentals.

## 🚀 Topics Covered
- HTML Document Structure
- Headings & Paragraphs
- Text Formatting
- Lists (Ordered & Unordered)
- Links
- Images
- Tables
- Forms
- Semantic HTML
- Audio & Video
- Iframes
- HTML Entities
- Comments
- Basic Project Examples

## 🛠️ Technologies Used
- HTML5
- Visual Studio Code

## 📂 Project Structure
```
HTML_basics/
│── index.html
│── images/
│── forms/
│── tables/
│── ...
```

## ▶️ How to Run
1. Clone the repository
```bash
git clone https://github.com/pari00720/HTML_basics.git
```

2. Open the project in VS Code.

3. Run any `.html` file using:
- Live Server Extension
- OR directly open it in your browser.

## 🎯 Purpose
The goal of this repository is to build a strong foundation in HTML before moving on to CSS, JavaScript, and modern frontend frameworks like React.
